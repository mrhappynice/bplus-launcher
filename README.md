# LaunchPad

A simple, self-hosted desktop app launcher built with **Rust** and **Vanilla JS**. 
It allows you to configure commands (like Docker containers or shell scripts) and provides a UI to launch them and open their web interfaces.

## Prerequisites

- [Rust](https://www.rust-lang.org/tools/install) installed.

## Setup & Run

1. **Build the backend:**
   ```bash
   cargo build --release
